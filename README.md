# Card Map Builder Pro

Draggable card maps for WordPress with images, captions, links, connections, an admin editor, and a frontend shortcode with zoom/pan/fullscreen.

This repository contains a single-file WordPress plugin: `cardmap.php`.

## Installation

1. Upload the `cardmap` folder to your WordPress `wp-content/plugins/` directory.
2. Activate the plugin from the WordPress admin.

## Development

- The main plugin file is `cardmap.php`.
- Edit the file locally and test inside a WordPress development environment.
